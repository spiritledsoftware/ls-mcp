# CI/CD Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add pnpm-based CI, Changesets release management, and GitHub Actions npm publishing with provenance.

**Architecture:** CI runs deterministic install and the existing quality gates on pull requests and `main`. Release automation uses Changesets to create version PRs and publish merged releases to npm.

**Tech Stack:** GitHub Actions, pnpm via Corepack, Node.js 22, Changesets, npm provenance.
